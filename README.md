# Bookstore – Plateforme de bibliothèque

Dans ce projet vous aurez pour tache de concevoir un website simulant une plateforme de bibliothèque. Elle offre aux lecteurs plusieux catégories de livres.

## Fonctionnalités principales

Nous avons l'espace : 
- shop
- post
- contact

## Objectifs

Moderniser la bibliothèque grâce au numérique. Aider les lecteurs à lire leurs sans se déplacer.
